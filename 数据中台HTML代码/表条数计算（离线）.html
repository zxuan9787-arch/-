参数详细描述如下:

   ●  源数据源：选择数据源(从数据源管理模块创建的数据源中选择)。

   ●  源数据库：选择数据源下的数据库。

   ●  源表：从源数据库中选择对应的数据表。

   ●  两表比较：统计两张表的数据然后进行比较。

   ●  满足成功条件的行数：配置该节点运行成功失败的条件。

   ●  比较数据库：当选择两表比较的时候，需要选择两个数据库。

   ●  比较表：被比较的表。

   ●  数值：该数值会和计算出来的结果进行比较。

   ●  自定义 SQL：勾选后就可以更灵活的配置 SQL 脚本。

   ●  使用变量替换：对SQL脚本中的变量进行替换。

   ●  在执行前清空结果行列表：上游节点运行结果里面可能保存的有上游节点的记录结果，勾选后将会清空。

   ●  在结果中添加：把这次查询的结果保存到运行结果中，下游节点可以使用该数据。

   ●  SQL 脚本：用户自定义的 SQL 脚本。
