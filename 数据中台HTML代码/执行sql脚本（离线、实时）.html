变量替换 ⓘ :

替换SQL脚本中的自定义变量

参数详细描述如下：

    ●  查询语句：支持SQL、MongoDB语句。

    ●  变量替换：SQL语句中支持引用变量。

        示例：delete from SRC where namel=${xx}，在这个SQL语句中，使用了变量xx，配置时需要勾选“变量替换”；注意：使用变量的时候格式为${变量名}。

    ●  INSERT 语句的结果字段：新增字段，表示INSERT语句的执行条数。

    ●  UPDATE 语句的结果字段：新增字段，表示UPDATE 语句的执行条数。

    ●  DELETE 语句的结果字段：新增字段，表示 DELETE语句的执行条数。

    ●  SELECT 语句的结果字段：新增字段，表示SELECT语句的执行条数。

    ●  添加参数：用上游的字段替换SQL 语句中的占位符(按照顺序依次替换)。

插件说明

转换过程中可以执行 SQL 脚本，该插件既可以单独使用，也可以配合上游插件使用。
